Min and Max Dates

You can set minimum and maximum dates by using {{ '{minDate: "date", maxDate: "date"}' }} options. The date format should match the dateFormat option. For example, if you are using {{ '{dateFormat: "YYYY-MM-DD"}' }}, the date should be in 'YYYY-MM-DD' format.
{{ '{minDate: {year: 2082, month: 6,day: 26}}' }}
Selected Date: {{ optionsMinDateValue }}
{{ '{dateFormat:"YYYY-MM-DD"}' }}
Selected Date: {{ optionsMaxDateValue }}
{{ '{dateFormat:"YYYY-MM-DD"}' }}
Selected Date: {{ optionsMinMaxValue }}

onSelect Callback (Event Binding)

You can use <rtc-nepali-datepicker> (dateChange)="onDateSelect($event)" </rtc-nepali-datepicker> to get the selected date. The date will be passed as a parameter to the function as DateObject. You can use this event to perform any action after selecting the date. Note that this is different than using the options object.
<rtc-nepali-datepicker> (dateChange)="onDateSelect($event)" </rtc-nepali-datepicker>
Selected Date: {{ optionsOnSelectValue }}

onClose Callback

You can use {{ 'onClose: (date: any) => {console.log("Datepicker closed");}' }} option to get the selected dates array when the datepicker is closed. The date array will be passed as a parameter to the function. You can use this option to perform any action after closing the datepicker.
{{ 'onClose: (date: any) => {console.log("Datepicker closed");}' }}
Selected Date: {{ optionsOnCloseValue }}

Disable Dates

You can disable specific dates by using {{ "{disableDates: ['YYYY-MM-DD', ...]}" }} option. You can also disable multiple dates by passing an array of dates. The date format should match the dateFormat option. For example, if you are using {{ "{dateFormat: 'YYYY-MM-DD'}" }} , the date should be in 'YYYY-MM-DD' format.
{{ '{disableToday:true}' }}
Selected Date: {{ optionsDisableTodayValue }}
{{ '{disableDates:["2082-6-32"]}' }}
Selected Date: {{ optionsDisableDateValue }}
{{ '{disableDates:["2082-6-1","2082-6-15","2082-6-28"]}' }}
Selected Date: {{ optionsDisableDatesValue }}
{{ '{disableDaysBefore:5}' }}
Selected Date: {{ optionsDisableBeforeValue }}
{{ '{disableDaysAfter:5}' }}
Selected Date: {{ optionsDisableAfterValue }}
{{ '{disableDaysBefore:5, disableDaysAfter:5}' }}
Selected Date: {{ optionsDisableBetweenValue }}
